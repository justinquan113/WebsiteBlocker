Privacy Policy

Blocked

A Chrome extension for blocking distracting websites. This page describes how Blocked handles your data.

What we collect

Nothing leaves your browser.

Blocked stores the following entirely on your local machine, using the browser's built-in chrome.storage.local API:

  • The list of websites you've chosen to block (always and scheduled)
  • Your schedule settings (days, start time, end time)
  • Your focus timer state (focus length, break length, cycle count, paused state)

What we don't collect

  • No analytics or usage tracking
  • No user accounts
  • No telemetry
  • No browsing history
  • No data sent to any server, ever

Third-party resources

The extension fetches a few static resources from third-party CDNs to render its UI:

  • Google Fonts for typography (Plus Jakarta Sans, Nunito)
  • Font Awesome for icons
  • Google favicon service to display the favicon next to each site in your block list and on the blocked page

These services may log standard request data (IP address, user-agent) according to their own privacy policies. We do not control what they log, and we never send identifying information to them beyond what's required to fetch the resource.

Permissions

The extension requests these Chrome permissions, used only for the purposes described:

  • storage — save your block list and settings locally
  • declarativeNetRequest — perform the actual site blocking at the network layer
  • tabs — read the current tab's URL so the popup can show the site you're on
  • alarms — drive the focus timer and check for schedule changes
  • notifications — alert you when a focus or break period ends
  • host_permissions <all_urls> — required so block rules can match any site you add

Removing your data

Uninstalling the extension removes every byte of data it stored. There is nothing to delete from any external service because nothing was ever sent there.

Changes

If this policy changes, the updated version will be posted here. The extension's version number in the Chrome Web Store will be incremented when material changes are made.
